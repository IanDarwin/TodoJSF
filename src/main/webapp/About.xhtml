About

This is a web application for maintaining small club membership lists. It has the ability to maintain a list at present; planned features include the ability to print lists, print labels, generate expiration reminders, etc.

Technology: The site is built using JavaServer Faces (JSF), Java Persistence Architecture (JPA), Apache DeltaSpike Data, and more. It should run on any standard Java EE 7 (and maybe 6) Application Server. Primarily developed and tested on JBoss WildFly.
